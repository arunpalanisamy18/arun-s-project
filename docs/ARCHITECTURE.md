# Architecture

This project is a modular astronomy research assistant.

## Frontend
- React + Vite + TypeScript
- TailwindCSS + shadcn/ui
- Plotly charts

## Backend
- FastAPI (Python)
- Endpoints for:
  - Light curve analysis
  - Spectra analysis
  - Transient detection
  - Survey mining

## Auth
- Frontend-only demo authentication using localStorage.
